# Rogue AI Visual Gallery

Mobile-friendly project for reviewing Rogue AI persona direction, cover prompts, and current reference assets.

Open `index.html` from OneDrive or a browser to view the gallery.

## Direction

Rogue AI should look human first: a Puerto Rican urbano artist trapped in a real body after a failed AI transfer. The AI should leak through subtle details only: eye glow, red diagnostic reflections, skin-level circuit scars, glitch artifacts, and posture.

## Avoid

- robot mascot
- exposed metal body
- superhero armor
- fantasy demon
- Terminator copy
- cheap hacker hoodie
- childish design
